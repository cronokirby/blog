Nets define Topologies

The standard definition of a topological space is in terms of open sets. We fix a collection T\mathcal{T}T of sets that are open by definition, and have a few laws about the structure of this collection.

One motivation for Topology is the study of sequences generalized as much as possible. You can define sequences very generally as functions: N→X\mathbb{N} \to XN→X for a space XXX, and study their convergence in this general setting.

One problem with sequences is that they're not sufficient to completely characterize topological spaces. For example, take the two following theorems:

If a space XXX is Hausdorff, then every sequence converges to at most one point.

If there is a sequence inside a subset SSS converging to sss, then sss is a limit point of SSS.

We'd like to see if these statements hold in the other direction, but unfortunately, sequences are too constraining here, and we cannot prove the converse of these statements generally. We need to make assumptions about the structure of the space if we want to prove these things

Nets

Nets generalize sequences, and allow us to more fully characterize topological spaces. A sequence is just a function N→X\mathbb{N} \to XN→X, giving us a linear sequence of points in the space:

Nets generalize this by allowing threads of points that can temporarily diverge, so long as we have some kind of forward direction:

Formally, a Net is a function J→XJ \to XJ→X, where JJJ is a Directed Set.

A directed set is a partially ordered set (J,→)(J, \to)(J,→) with the following extra property:

For any points a,b∈Ja, b \in Ja,b∈J, there exists c∈Jc \in Jc∈J, with a→ca \to ca→c and b→cb \to cb→c. We call this being forward directed.

A partial order →\to→ on a set is a relation satisfying:

  • ∀a.a→a\forall a. \quad a \to a \quad∀a.a→a (Reflexivity)

  • ∀a b c.a→b, b→c⟹a→c\forall a\ b\ c. \quad a \to b,\ b \to c \implies a \to c \quad∀a b c.a→b, b→c⟹a→c (Transitivity)

The crucial property of being forward directed means that we can't have multiple "tails" diverging off:

instead, these tails always meet back eventually. This is because we can take any two points on these different tails, and there will be some point that succeeds both of them.

Formally, given a Net f:J→Xf : J \to Xf:J→X, a tail is a set of the form:

f(a→):={f(b) ∣ b∈J, a→b}f(a_\to) := \{f(b) \ |\ b \in J,\ a \to b \}f(a→​):={f(b) ∣ b∈J, a→b}

This consists of all points in the net that come after a given position.

Examples of Directed Sets

It's clear that N\mathbb{N}N is a directed set, under ≤\leq≤, so sequences are also Nets.

Another good example of a directed set consists of a collection C\mathcal{C}C of sets that is closed under pairwise intersection. We have A→BA \to BA→B precisely when A⊇BA \supseteq BA⊇B. This is forward directed, since given AAA and BBB, we can take A∩BA \cap BA∩B as their common successor. Because the set C\mathcal{C}C is closed under pairwise intersection, this is also an element of the collection.

We can generalize this to collections of sets where for any A,B∈CA, B \in \mathcal{C}A,B∈C, there exists D∈CD \in \mathcal{C}D∈C with:

A∩B⊇DA \cap B \supseteq DA∩B⊇D

An example of such a collection would be a basis for a given topology.

Convergence

With this in hand, we can define convergence of a net.

A net f:J→Xf : J \to Xf:J→X converges to a point x∈Xx \in Xx∈X precisely when every neighborhood of xxx contains a tail of fff.

More concretely, this says that for every open set NxN_xNx​ with x∈Nxx \in N_xx∈Nx​, there exists a position aaa, such that for all a→ba \to ba→b, f(b)∈Nxf(b) \in N_xf(b)∈Nx​.

Nets allow us to work in this general setting though.

Limit Points

As a reminder, a limit point of a set SSS is a point xxx such that every neighborhood of xxx intersects SSS.

Nets allow us to more precisely capture the nature of limit points better than sequences.

A point xxx is a limit point of SSS precisely when there exists a net in SSS converging to xxx.

⟸\impliedby⟸

(This direction works fine with sequences)

Every neighborhood of xxx contains a tail in SSS (by convergence), and thus a point in SSS.

⟹\implies⟹

(This direction doesn't work, since we can't always summon a sequence with the desired properties)

Take the directed set of neighborhoods N(x)\mathcal{N}(x)N(x). Because each neighborhood of xxx intersects with SSS, we can define a Net f:N(x)→Sf : \mathcal{N}(x) \to Sf:N(x)→S. This Net obviously converges to xxx, and is contained within SSS.

□\square□

Compactness

In a metric space, compactness is equivalent to every sequence having a convergent subsequence.

We can generalize this beyond metric spaces using Nets, but we first need to define sub-Nets.

Sub Nets

With sequences, the idea is that we take an increasing subset of the indices that never ends, and use that as our subsequence. With Nets, we need to take a directed subset, and also make sure that it never ends.

Given directed sets JJJ and III, a cofinal map is a function g:I→Jg : I \to Jg:I→J, such that:

  • a→b⟹g(a)→g(b)a \to b \implies g(a) \to g(b)a→b⟹g(a)→g(b)
  • For every α∈J\alpha \in Jα∈J, there exists a∈Ia \in Ia∈I such that α→g(a)\alpha \to g(a)α→g(a)

The first property just says that the inclusion from III to JJJ can't completely obliterate all the properties of a directed set, which would give us subnets with a completely different structure.

The second property prevents us from "cheating" by taking an infinite net, and taking just a finite subnet. There always elements in the subnet succeeding us.

Concretely, a subnet of f:J→Xf : J \to Xf:J→X is the composition gf:I→Xgf : I \to Xgf:I→X of fff with a cofinal map g:I→Jg : I \to Jg:I→J.

Compactness and SubNets

Equipped with this key connection between accumulation points and convergent subsequences, we can prove that a subspace C⊆XC \subseteq XC⊆X is compact precisely when every net in CCC has a convergent subnet.

As a reminder, a subspsace CCC is defined to be compact when every collection of open sets covering CCC has a finite sub-collection covering CCC.

An equivalent characterization of compactness is:

A subspace CCC is compact precisely when for every collection of closed sets with the finite intersection property, the intersection of the collection is non-empty.

A collection has finite intersection property iff finite intersections of its sets are non-empty.

Proof:

⟹\implies⟹

We assume that our space is compact, and we consider an arbitrary net f:J→Cf : J \to Cf:J→C.

Equivalence with a Topology

So, we've seen that nets are sufficient to characterize quite a few topological properties. The cool thing about nets is that you can construct nets that mimic the structure inherent to any topological space.

In fact, knowing which nets converge is a complete characterization of a topological space.

A criterion is a mapping from nets fff and points x∈Xx \in Xx∈X to truth values Ω(f,x)∈{0,1}\Omega(f, x) \in \{0, 1\}Ω(f,x)∈{0,1}, such that Ω(f,x)⟹Ω(gf,x)\Omega(f, x) \implies \Omega(gf, x)Ω(f,x)⟹Ω(gf,x), for any subnet gfgfgf.

Evidently, a topological space induces a criterion, because convergence is defined for any topological space.

Induced Topology

More surprisingly, we can use a criterion to induce a topology on XXX.

First, note that we can equivalent define a topology in terms of its closed sets. We require that ∅\emptyset∅ and XXX are closed, the the finite union of closed sets is closed, and finally, that the arbitrary union of closed sets is closed. We can go from a closed topology to an open topology by taking the complements of sets.

A criterion Ω\OmegaΩ induces a closed topology on XXX, by defining a set CCC to be closed precisely when for every net fff in CCC we have

Ω(f,x)⟹x∈C\Omega(f, x) \implies x \in CΩ(f,x)⟹x∈C

It's common knowledge that closed sets contain their limit points, and this characterization of closed sets inspires this definition.

We now show that this is a closed topology.

Now, since every xxx is in XXX, then XXX is obviously closed.

Now, Ω(f,x)\Omega(f, x)Ω(f,x) is always false if x∈∅x \in \emptysetx∈∅, since it's impossible to even ask this question.

We could also define open sets directly, by saying that a set UUU is open precisely when no net outside of UUU can converge to a point inside of UUU. This is just a reformulation of what we've said so far.

Back and Forth

One remaining question is whether or not applying this construction twice gives you the same result.

Topology first

A topology T\mathcal{T}T gives us a criterion Ω\OmegaΩ for convergence to a point. Does the induced topology TΩ\mathcal{T}_{\Omega}TΩ​ equal T\mathcal{T}T?

Yes, simply because a closed set is completely characterized by the fact that it contains its limit points. We've shown that a limit point is just a point with a net converging to it. Our induced topology marks every set containing its limit points as closed, which matches are original topology exactly.

Criterion first

Now, if you have a criterion Ω\OmegaΩ, then you have an induced criterion: Ω′\Omega'Ω′ by considering the usual convergence, but under the induced topology TΩ\mathcal{T}_\OmegaTΩ​.

It's pretty easy to show that Ω(f,x)⟹Ω′(f,x)\Omega(f, x) \implies \Omega'(f, x)Ω(f,x)⟹Ω′(f,x), but I haven't yet been able to prove the converse.

It seems that in Kelley's General Topology, a few more axioms are required, and they might be able to patch this up.

Generalizations

The formulation of a topology in terms of a criterion invites generalizations. For example, you might relax the forward directedness and see what wacky spaces come out. You might also strengthen the conditions on the indexing set, and see what kind of strong properties of the space arise.
