Abstract

Determining whether an arbitrary access structure can be realized by an ideal linear secret sharing scheme is an important research topic. we use linear codes as the main tool to construct matrices HHH and GGG over a finite field Fq\mathbb{F}_qFq​ for a given access structure Γ\min\Gamma_{\min}Γmin​, and show that a necessary and sufficient condition for the existence of an ideal linear secret sharing scheme realizing Γ\min\Gamma_{\min}Γmin​ is that the equation GHT=0GH^{\mathsf{T}}=0GHT=0 has a solution. If this equation has a solution, then HHH serves as the parity-check matrix of a linear code that realizes Γ\min\Gamma_{\min}Γmin​, and GGG is the corresponding generator matrix. Furthermore, we prove that the result is equivalent to the following statement: there exists an ideal linear code for realizing the Γ\min\Gamma_{\min}Γmin​ if and only if it is the port of a matroid that is representable over Fq\mathbb{F}_qFq​.
