Topological Groups

If GGG is a Topological Group, then the connected component CCC containing the identity eee is a normal subgroup of GGG.

First, note that for any g∈Gg \in Gg∈G, gCgCgC is also a connected component. If gx,gy∈gCgx, gy \in gCgx,gy∈gC, then x,y∈Cx, y \in Cx,y∈C. This means they share a connected subspace AAA. The image gAgAgA is also a connected subspace, since the action of ggg is a continuous function. Evidently, gAgAgA contains gxgxgx and gygygy. This means that any two points of gCgCgC share a connected subspace.

A similar proof shows that CgCgCg is also a connected component.

It's easy to see that CCC is a subgroup.

First, e∈Ce \in Ce∈C, by assumption.

Second, we need to show that a,b∈C⟹ab−1∈Ca, b \in C \implies ab^{-1} \in Ca,b∈C⟹ab−1∈C. Because e∈Ce \in Ce∈C, we have ab−1∈ab−1Cab^{-1} \in ab^{-1}Cab−1∈ab−1C. Since a=ab−1ba = ab^{-1}ba=ab−1b, and b∈Cb \in Cb∈C, we have a∈ab−1Ca \in ab^{-1}Ca∈ab−1C as well. But since ab−1Cab^{-1}Cab−1C is a connected component, i.e. equivalence class, and it contains aaa, it must be equal to CCC, since CCC is an equivalence class containining aaa. Since ab−1C=Cab^{-1}C = Cab−1C=C, ab−1ab^{-1}ab−1 must be in CCC.

To see that CCC is normal, we show that gC=CggC = CggC=Cg. For this, note that gCgCgC and CgCgCg are both connected components, as we showed earlier. But, since they both contain ggg, they must be equal.
